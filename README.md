## [WISP](http://jeditoolkit.com/wisp/) - Homoiconic JS with clojure syntax, s-expressions & macros for Meteor framework.

----

**Usage:** `meteor add nsmeta:wisp`

**Version:** `0.10.0_1`

* Supports source-maps
* Experimental support for namespaces/require
